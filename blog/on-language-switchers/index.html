On language switchers
May 12, 2017
1 minute read

http://www.flagsarenotlanguages.com/blog/best-practice-for-presenting-languages/ http://www.flagsarenotlanguages.com/blog/why-flags-do-not-represent-language/ https://usersnap.com/blog/design-language-switch/

  • Don’t use flags. Multiple languages per country. (Examples)

I think https://github.com/newscloud/yii2-language-switcher/ is very wrong. It translates languages list while it should not. Every language should be listed either using its native name or English name.

For example, I’ve accidentally switched to Chinese. How would I switch back to Russian? How would I know that I should select 俄语 instead of 英语?
